# Codemao Cloud（点猫云）

> 基于Python语言的图形化加密云盘软件 适用于较小文件的远程(加密)分享
> 文件托管服务基于Codemao（点猫科技）服务器

---

## 项目简介

Codemao Cloud（点猫云）是一个用于上传和下载编程猫云端文件的桌面应用程序。
该项目提供了图形界面，方便用户进行文件管理和云端数据传输。

---

### 1.工作模式与使用场景

#### Data模式：

适用于小文件（16mb以内且以文本、图片、视频等简单格式存储）的远程分享，无加密功能，使用任何浏览器访问链接即可在线查看或下载。

#### XML模式：

适用于任何格式的文件（文件大小不超过45mb）的远程(私密)分享，可自行设定密码进行加密，仅能通过点猫云软件下载模式进行解析和存储。

---

### 2.功能实现与代码执行流程

项目基于Python语言
下载页面与主页面的图形化实现基于Tkinter
上传页面的图形化实现基于Pyqt5（V2.0已使用Tkinter进行重构）

#### 上传模式流程：

1. Data模式：获取文件链接—以二进制格式读取文件内容—Request库进行提交—生成链接—跳转链接
2. XML模式：
    * （V1.0）获取文件链接与密码—以二进制/字符串格式读取文件内容—对密码进行SHA1加密—文件内容转换为base64格式—将文件名、密码、文件内容填充至XML—Request库进行提交—生成链接—跳转链接
    * （V2.0）获取文件链接与密码—以二进制/字符串格式读取文件内容—密码转换为密钥—进行AES-256加密—密文转换为base64格式输出—将文件名、密文填充至XML—强制编码为UTF-8—Request库进行提交—生成链接—跳转链接

#### 下载模式流程：

XML模式：
* （V1.0）获取云盘链接、密码与保存路径—Request库抓取信息—校验是否为有效链接—XML格式解析—对新密码进行SHA1加密，校验密码是否与云盘密码一致—文件内容base64解码—写入文件并保存
* （V2.0）获取文件链接、密码与保存路径—Request库抓取信息—校验是否为有效链接—XML格式解析—密码转换为密钥—密文base64解码—进行AES-256解密—写入文件并保存

## 依赖环境与文件说明

依赖环境：

```
Python 3.x
PyQt5（V2.0已使用Tkinter进行重构）
Requests
xmltodict
pycryptodome
```

文件说明：

| 文件 | 功能描述 |
|------|----------|
| `index.py` | 应用程序主入口，具有导览与索引功能 |
| `up.py` | 使用Tkinter重构的上传模式程序，默认上传程序|
| `down.py` | 基于Tkinter的下载模式程序 |
| `up_pyqt5.py` | 基于Pyqt5的上传模式程序，非默认上传程序 |
| `format.xml` | XML格式填充模版 |
| `LICENSE` | Apache-2.0开源协议 |
| `README.md` | 项目介绍 |

## 注意事项

1. 本软件并不适合大型文件的安放，适用于小型文件的传播，例如分享照片，代码，视频等。
2. 本软件私密化程度极高，暴力破解密码用时不少于3.5小时
3. 本软件下载端链接请填写完整链接，即https://static.box3.codemao.cn/block/......
4. 切勿过分上传文件，切勿过度使用点猫服务器
5. 使用本软件进行制作，再开发，使用的一切行为与后果和开发者无关
6. 云盘链接为永久性链接，但丢失概不负责，建议用于短期(加密)分享而非长期固定存档
7. 云盘链接具有私密性，Data模式传输请保护好链接，有链接者即可直接访问
8. 云盘空间有限，Data模式文件大小不能超过18mb，XML模式文件大小不可超过45mb
9. 本项目仅供学习交流使用，请遵守相关法律法规和编程猫平台的服务条款。
